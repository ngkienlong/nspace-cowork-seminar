Phần 1

Phương trình bậc nhất hai ẩn

Định nghĩa

Phương trình bậc nhất hai ẩn là gì?

Dạng tổng quát

ax + by = c Trong đó: • a, b, c là số đã biết • a ≠ 0 hoặc b ≠ 0 • x, y là hai ẩn số

Ví dụ

✅ 4x + 3y = 5 (a=4, b=3) ✅ 0x + y = −1 (a=0, b=1) ❌ 0x + 0y = 3 (a=b=0 → không phải!)

Khái niệm

Nghiệm của phương trình

Định nghĩa

Cặp số (x₀; y₀) là nghiệm nếu khi thay vào phương trình ta được khẳng định ĐÚNG.

Ví dụ: 4x + 3y = 5

• (2; −1) → 4·2 + 3·(−1) = 8 − 3 = 5 ✅ • (1; 0) → 4·1 + 3·0 = 4 ≠ 5 ❌

Tìm nghiệm bằng bảng giá trị

Phương trình: x + 2y = 5 → y = (5 − x)/2

x−2−1012
y = (5−x)/23,532,521,5

→ Mỗi giá trị x cho 1 nghiệm → Vô số nghiệm!

Tính chất quan trọng
∞

Mỗi phương trình bậc nhất hai ẩn đều có VÔ SỐ nghiệm

📐 Biểu diễn hình học: Nghiệm = Đường thẳng

Phương trình x + 2y = 3

→ y = −0,5x + 1,5

  • Điểm A(0; 1,5) ✅
  • Điểm B(3; 0) ✅
  • Mọi điểm trên đường thẳng d đều là nghiệm

Tập nghiệm = Đường thẳng d: ax + by = c

📐 Các trường hợp đặc biệt

y = −2 (0x + y = −2)

x = 3 (x + 0y = 3)

Phần 2

Hệ hai phương trình bậc nhất hai ẩn

Định nghĩa

Hệ hai phương trình bậc nhất hai ẩn

Dạng tổng quát

⎧ ax + by = c ⎨ ⎩ a'x + b'y = c' Gồm 2 phương trình bậc nhất hai ẩn ghép lại.

Nghiệm của hệ

Cặp (x₀; y₀) là nghiệm của hệ nếu nó ĐỒNG THỜI là nghiệm của CẢ HAI phương trình.

🎯 Ý nghĩa hình học: Nghiệm = Giao điểm

Hệ phương trình:

⎧ 2x − y = 0

⎩ x + y = 3

Kiểm tra (1; 2):

  • 2·1 − 2 = 0 ✅
  • 1 + 2 = 3 ✅

→ Nghiệm: (1; 2) = Giao điểm M

📌 Tổng kết

Ghi nhớ kiến thức trọng tâm

  • PT bậc nhất hai ẩn: ax + by = c (a ≠ 0 hoặc b ≠ 0) → có VÔ SỐ nghiệm
  • Tập nghiệm biểu diễn hình học là một ĐƯỜNG THẲNG
  • Hệ hai PT bậc nhất hai ẩn: ghép 2 PT → nghiệm là cặp số thoả mãn CẢ HAI
  • Ý nghĩa hình học: nghiệm của hệ = GIAO ĐIỂM hai đường thẳng
  • Luôn kiểm tra nghiệm bằng cách THAY vào cả hai phương trình
