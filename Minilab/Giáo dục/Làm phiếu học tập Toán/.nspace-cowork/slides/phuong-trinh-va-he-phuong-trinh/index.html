02
Định nghĩa

Phương trình bậc nhất hai ẩn

  • Dạng tổng quát: ax + by = c (a ≠ 0 hoặc b ≠ 0)
  • x, y là hai ẩn; a, b, c là các hệ số đã biết
  • Ví dụ: 2x − 3y = 5 là phương trình bậc nhất hai ẩn
03
Khái niệm

Nghiệm của phương trình

Định nghĩa

Cặp số (x₀; y₀) là nghiệm nếu khi thay vào phương trình ta được đẳng thức đúng: ax₀ + by₀ = c

Ví dụ

Phương trình x + 2y = 3: • (1; 1) là nghiệm vì 1 + 2·1 = 3 ✓ • (3; 0) là nghiệm vì 3 + 2·0 = 3 ✓

04
Tính chất quan trọng

Số nghiệm & biểu diễn hình học

  • Mỗi phương trình bậc nhất hai ẩn có VÔ SỐ nghiệm
  • Tập nghiệm biểu diễn bởi một ĐƯỜNG THẲNG trên mặt phẳng tọa độ
  • Đường thẳng đó gọi là đường thẳng ax + by = c
05
Ví dụ minh hoạ

Các trường hợp đặc biệt

a = 0 → dạng y = k

0x + y = −2 → y = −2 Nghiệm: (x; −2) với mọi x → Đường thẳng song song trục Ox

b = 0 → dạng x = k

x + 0y = 3 → x = 3 Nghiệm: (3; y) với mọi y → Đường thẳng song song trục Oy

Định nghĩa

Hệ hai phương trình bậc nhất hai ẩn

  • Gồm hai phương trình bậc nhất hai ẩn ghép lại
  • Dạng: { ax + by = c ; a'x + b'y = c' }
  • Nghiệm của hệ: cặp số đồng thời thoả cả hai phương trình
08
Ý nghĩa hình học

Nghiệm của hệ = Giao điểm hai đường thẳng

Cắt nhau → 1 nghiệm

Hai đường thẳng cắt nhau tại đúng 1 điểm → Hệ có nghiệm duy nhất

Song song → vô nghiệm

Hai đường thẳng song song → Không có giao điểm → Hệ vô nghiệm

Bài 2

Tìm nghiệm của 2x − y = 1

Yêu cầu

Hoàn thành bảng giá trị: x = −1, −0.5, 0, 0.5, 1, 2 y = 2x − 1 = ?

Nghiệm tổng quát

y = 2x − 1 Nghiệm: (x; 2x−1) với mọi x ∈ ℝ

12
Bài 3

Viết nghiệm & biểu diễn hình học

  • a) 2x − y = 3 → y = 2x − 3
  • b) 0x + 2y = −4 → y = −2 (đường ngang)
  • c) 3x + 0y = 5 → x = 5/3 (đường dọc)
13
Bài 4

Kiểm tra nghiệm của hệ phương trình

Hệ phương trình

{ 2x = −6 5x + 4y = 1 } → Có phải hệ PT bậc nhất hai ẩn?

Kiểm tra (−3; 4)

• 2·(−3) = −6 ✓ • 5·(−3) + 4·4 = −15 + 16 = 1 ✓ → (−3; 4) là nghiệm của hệ

14
Bài 5

Tìm nghiệm chung của hệ

  • Cho 5 cặp số: (−2;1), (0;2), (1;0), (1.5;3), (4;−3)
  • PT (1): 5x + 4y = 8
  • PT (2): 3x + 5y = −3
  • Tìm cặp thoả cả (1) và (2) → nghiệm của hệ
